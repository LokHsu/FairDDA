# FairDDA

This repository contains source codes and datasets for the paper:

- Improving Recommendation Fairness via Graph Structure and Representation Augmentation

## Usage
### Train

- Training FairDDA on MovieLens:
```shell
python fairness_gcn_movielens_aug_hisc.py
```

- Training FairDDA on IJCAI15:
```shell
python fairness_gcn_ijcai_aug_hisc.py
```

- Training FairDDA on LastFM:
```shell
python fairness_gcn_lastfm_aug_hisc.py
```
